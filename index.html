MyBlackJack Pro

HTML 5 / CSS 3 / JavaScript

BlackJack is a popular fast-paced betting game. Each round sets the player against the dealer. First to acheive a score of 21 (or as close to it possible) wins the game. Score is tabulated by summing the values of cards dealt during each round. Additional features include Split, Double down, and Insurance. Professional players can also activate the real-time counting option to practice their card-counting skills.

Exclusive Cars
For My Garage

Python / Django / PostgreSQL

Exclusive Cars For My Garage (ECFMG) is a Cars Sales website. Clients can browse used supercars and choose from two sections, featured vehicles and informed buyer(IB). The IB category contains vehicles with a salvage title and is especially geared towards car enthusiasts. Customers can use the search feature to search for a car by make and then further filter their results. Vehicle lisings contain an impressive image gallery; prospective clients send inquiries to sellers. An inquiry is stored on the users dashboard and displayed on login (Facebook and Google OAuth).
